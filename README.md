## Installation

1. Put the file **crauEmu.py** in same location as [**uEmu.py**](https://github.com/alexhude/uEmu/blob/master/uEmu.py).
2. Use `File / Script file...` or `ALT+F7` in IDA to load **crauEmu.py**
